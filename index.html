Typography

The different text sizes used throughout the site.

The font used throughout the site is Inter. The font is a sans-serif font that is easy to read and has a modern look.

The font is available in 10 weights, from 100 to 900. The fallback font is sans-serif.

Main font size is 16px. Subtext font size is 14px, but should not use a light gray font color for accessibility reasons.
