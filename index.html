Typography

The different text sizes used throughout the site.

The font used for content is Inter, for headings of all sizes we use Alexandria. The font is a sans-serif font that is easy to read and has a modern look.

The font is available in 10 weights, from 100 to 900. The fallback font is sans-serif.

Main font size is 16px. Subtext font size is 14px, but should not use a light gray font color for accessibility reasons.

Headings

Please note: Headings use Alexandria font instead of Inter, this means headers should be created using the proper tags i.e. h1, h2, etc and not just styled with a in the editor or you are very likely to have Inter be used

Link
More about Robert Foster

Link color is ACC-Purple. Hover color is ACC-Dark-Purple. Underline is needed to indicate that it is a link (or other visual effect). For body text make the link 200 font weight heavier, for footer or headers where all elements are expected to be a link, you may use the default body weight (e.g. 400)
